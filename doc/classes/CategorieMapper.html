CategorieMapper

Extends \dice\mapper\SQLMapper

Description of CategorieMapper

La Classe SQLMapper est un DataMapper pour une base SQL Les objets mappés doivent implanter l'interface iDataMappable Ce mapper est simpliste : il ne mappe pas correctement pas les objets contenant un tableau et lorsque un objet contient une référence vers un autre objet le mapper mappe en utilisant l'oid de l'objet référencé
